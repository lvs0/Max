# Max Dashboard

Un dashboard Jarvis-like avec glassmorphisme, orb animé et particules.

## Features

- **Glassmorphisme** - Interface translucide avec effets de flou
- **Orb animé** - Sphère centrale bleu→jaune avec pulsation et rotation
- **Particules** - Animation de particules flottantes en arrière-plan
- **Sidebar** - Gestion des tâches, planning et notes
- **Chat bar** - Interface de commande conversationnelle

## Démo

Ouvrir `index.html` dans un navigateur moderne.

## Design

- Gradient bleu nuit profond (#1a1a2e → #0f3460)
- Accents bleu cyan (#4facfe) et or (#f6d365)
- Typographie Space Grotesk + Inter
- Animations CSS fluides
